## Tentang

Repository ini merupakan bagian dari project [Algokata](https://github.com/fcrezza/algokata) yang digunakan untuk menjalankan dan melakukan validasi kode dari user.

## Getting Started

### Prerequisite

- npm

```
npm install npm@latest -g
```

### Instalasi

1. Clone repository
```
git clone https://github.com/fcrezza/algokata.git
```
2. Install NPM packages
```
npm install
```
3. Salin environment variabel
```
cat .env.example > .env
```
4. Jalankan server lokal
```
npm run dev
```
